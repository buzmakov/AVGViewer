# AVG Viewer

Приложение для просмотра файлов самописца формата AVG/MPH/MIN с метками Marks.

## Установка

```bat
python -m venv .venv
.venv\\Scripts\\python -m pip install -r requirements.txt
```

## Запуск

```bat
.venv\\Scripts\\python main.py
```

Можно сразу передать AVG-файл:

```bat
.venv\\Scripts\\python main.py "..\\клонидин мкт\\21_2025_клонидин мкт_001.AVG"
```

## Возможности

- Открытие `.AVG` файла.
- Автоматический поиск связанных `_Marks.txt`, `_001.MPH`, `.MIN`, `_Config_Info.txt`.
- Отображение выбранных параметров через чекбоксы.
- Вертикальные линии событий из Marks.
- Ось времени и статус-бар отображают время в формате `HH:MM:SS`.
- Масштабирование по времени колесом мыши: колесо вверх приближает вокруг курсора, колесо вниз отдаляет.
- Сопоставимый масштаб кривых: видимые серии отображаются как нормированное отклонение (`z-score`), чтобы скачки были видны на всех кривых независимо от физических единиц.
- Выбор окна усреднения отображаемых данных: `1`, `2`, `5`, `10`, `60` секунд.
- Выделение диапазона мышью на графике.
- Копирование значений видимых серий по выделению в буфер обмена в TSV-формате кнопкой или через `Ctrl+C`.
- При копировании используются текущие усреднённые данные, но в физических значениях, не в `z-score`.
- Наложение поударных MPH-данных на выделенный участок.

## Особенности формата

- Текстовые файлы читаются в кодировке Windows-1251 (`cp1251`).
- Таблицы используют `Tab` как разделитель и запятую как десятичный разделитель.
- В AVG/MPH строках могут встречаться встроенные текстовые метки после последнего числового поля; парсер игнорирует лишние поля после ожидаемых столбцов.
