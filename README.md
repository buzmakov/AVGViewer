# AVG Viewer

Приложение для просмотра файлов самописца формата AVG/MPH/MIN с метками Marks.

## Установка

```bat
python -m venv .venv
.venv\Scripts\python -m pip install -r requirements.txt
```

## Запуск

```bat
.venv\Scripts\python main.py
```

Можно сразу передать AVG-файл:

```bat
.venv\Scripts\python main.py "..\клонидин мкт\21_2025_клонидин мкт_001.AVG"
```

## Сборка exe для Windows

Для сборки одного исполняемого файла используется PyInstaller. Команды нужно выполнять из папки проекта `avg-viewer`.

1. Создать виртуальное окружение и установить зависимости приложения:

```bat
python -m venv .venv
.venv\Scripts\python -m pip install -r requirements.txt
```

2. Установить PyInstaller:

```bat
.venv\Scripts\python -m pip install pyinstaller
```

3. Собрать приложение в один exe-файл без консольного окна:

```bat
.venv\Scripts\python -m PyInstaller --noconfirm --onefile --windowed --name AVGViewer main.py
```

После успешной сборки готовый файл будет находиться здесь:

```txt
dist\AVGViewer.exe
```

Примечания:

- При первом запуске `AVGViewer.exe` может открываться несколько секунд: one-file exe распаковывает Python-зависимости во временную папку.
- Папки `build` и `dist`, а также файл `AVGViewer.spec`, создаются PyInstaller автоматически.
- Если нужно пересобрать exe после изменений в коде, достаточно повторно выполнить команду сборки из пункта 3.

## Возможности

- Открытие `.AVG` файла.
- Автоматический поиск связанных `_Marks.txt`, `_001.MPH`, `.MIN`, `_Config_Info.txt`.
- Отображение выбранных параметров через чекбоксы.
- Вертикальные линии событий из Marks.
- Список меток слева с переходом к выбранной метке кликом.
- Ось времени и статус-бар отображают время в формате `HH:MM:SS`.
- Масштабирование по времени колесом мыши: колесо вверх приближает вокруг курсора, колесо вниз отдаляет.
- Горизонтальная полоса прокрутки по времени после увеличения масштаба.
- Выделение диапазона мышью на графике.
- Выбор окна усреднения для копирования данных: `1`, `2`, `5`, `10`, `60` секунд.
- Копирование значений видимых серий по выделению в буфер обмена в TSV-формате кнопкой или через `Ctrl+C`.
- Усреднение влияет только на копируемые данные; график остаётся в исходном разрешении AVG.
- Наложение поударных MPH-данных на выделенный участок.
- Повторное нажатие кнопки MPH скрывает наложенные MPH-данные.

## Особенности формата

- Текстовые файлы читаются в кодировке Windows-1251 (`cp1251`).
- Таблицы используют `Tab` как разделитель и запятую как десятичный разделитель.
- В AVG/MPH строках могут встречаться встроенные текстовые метки после последнего числового поля; парсер игнорирует лишние поля после ожидаемых столбцов.
